# AWS Exams Study

This repository contains organized study notes and summaries for the AWS Certified Cloud Practitioner exam.


## Contents

- [practitioner/concepts.md](practitioner/concepts.md): Key AWS concepts in English, grouped by category, with concise explanations and bolded keywords for exam preparation.
- [practitioner/conceptos_es.md](practitioner/conceptos_es.md): Spanish translation of the key AWS concepts, with the same structure and focus.
- [practitioner/services.md](practitioner/services.md): Detailed AWS services descriptions in English, grouped by category, with exam-focused tips.
- [practitioner/services_es.md](practitioner/services_es.md): Spanish translation of AWS services, with the same structure and exam context.

## Usage

Use these files to review and reinforce your understanding of AWS foundational topics. Each section is designed to help you quickly recall the most important points for the exam.
